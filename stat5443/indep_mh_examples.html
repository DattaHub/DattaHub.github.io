Independence Sampler

  • We want to use an independence sampler: \[ g( \theta \mid \theta^{(t)}) = g(\theta) \]

  • The point we are trying to argue that Independence samplers can be very good or very bad.

  • Good independence samplers have heavier tails than the target.

Example 1

  • Normal likelihood, Cauchy Prior, (Independent) Cauchy proposal.
target <- function(theta,n=20,ybar=0.0675){
  dens <- exp(-n*(theta - ybar)^2/2)*(1/(1+theta^2))
  return(dens)
}

proposal <- function(theta_new, theta_old){
  dens <- dcauchy(theta_new,0,1) 
}

Aceptance Probability

accept_prob <- function(theta_new,theta_old){
  alpha_mh <- log(target(theta_new)) + log(proposal(theta_old, theta_new))
  - log(target(theta_old)) - log(proposal(theta_new, theta_old))
  alpha_mh = exp(alpha_mh)
  alpha_mh <- ifelse(alpha_mh>1,1,alpha_mh)
  return(alpha_mh)
}

The MH Sampler

MH_sampler <- function(theta_init, niter){
  theta_samples <- rep(0,niter)
  theta_samples[1] <- theta_init 
  for(i in 1:niter){
    theta_old = theta_samples[i]
    theta_new <- rcauchy(1)
    u <- runif(1)
    alpha_mh = accept_prob(theta_new,theta_old)
    if(u <= alpha_mh){
      theta_samples[i+1] = theta_new
    }else{
      theta_samples[i+1] = theta_old
    }
  }
  return(theta_samples) 
}

Now Run the MH

set.seed(453)
theta.init = 0 # initial value of theta
nmc = 1e5
burn.in = nmc/2
th.samp = MH_sampler(theta.init,nmc)
mean(th.samp[(burn.in+1):nmc])
## [1] 0.06036898

Trace plot

plot(th.samp,type="l")

Now the Poor Example

  • I want another independence sampler with proposal function: \(N(y,1)\)
  • The only two things to change: proposal function and the MH algo.
proposal <- function(theta_new, theta_old, ybar=0.0675){
  dens <- dnorm(theta_new,ybar,1)
}

Change the MH algo too

MH_sampler <- function(theta_init, niter, ybar = 0.0675){
  theta_samples <- rep(0,niter)
  theta_samples[1] <- theta_init 
  for(i in 1:niter){
    theta_old = theta_samples[i]
    theta_new <- rnorm(1,ybar,1)
    u <- runif(1)
    alpha_mh = accept_prob(theta_new,theta_old)
    if(u <= alpha_mh){
      theta_samples[i+1] = theta_new
    }else{
      theta_samples[i+1] = theta_old
    }
  }
  return(theta_samples) 
}

Choice of Initial Values

  • We will use two initial values: 0 and 10.
  • The true mean is around 0.0620.
set.seed(453)
theta_init_good = 0
theta_init_poor = 10
th.samp.good = MH_sampler(theta_init_good,1e5)
th.samp.poor = MH_sampler(theta_init_poor,1e5)

mean(th.samp.good)
## [1] 0.05584084
mean(th.samp.poor)
## [1] 10
